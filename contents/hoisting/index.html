Hoisting

Variable declarations and functions declarations are process before any code is executed, so declare a variable o function anywhere in the code is equivalent to the declare it at the top.

Sample 1

In this code, the first line is printing 'undefined' because the hoisting only puts the declaration of the variable at the top, but the initialization keeps at the same line of code.

console.log(definedLater); // undefinded var definedLater; definedLater = 'I am defined';
